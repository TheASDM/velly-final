SETUP INSTRUCTIONS
==================

1. Create the directory structure:
   mkdir -p dnd-campaign-site/{nginx/conf.d,chatbot,campaign-data,static-site/{css,js,data,resources/maps},ssl}

2. Download all files from this artifact

3. Rename and move files to match the structure above:
   - Remove the prefixes (e.g., nginx-nginx.conf → nginx/nginx.conf)
   - Place each file in its proper directory

4. Copy .env.example to .env:
   cp .env.example .env

5. Edit .env and add your Anthropic API key

6. Make deploy.sh executable:
   chmod +x deploy.sh

7. Run deployment:
   ./deploy.sh

8. Access your site at http://localhost:8080
